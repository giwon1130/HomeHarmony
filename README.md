# HomeHarmony

아파트 선택 과정에서 입지와 생활 편의 요소를 함께 비교할 수 있도록 돕는 주거 선택 지원 서비스입니다.

## 프로젝트 소개
- 주거지를 선택할 때 필요한 다양한 정보를 한 번에 비교할 수 있도록 정리한 개인 프로젝트
- 단순 매물 조회보다 생활 편의와 선택 기준 비교에 초점을 맞춤

## 핵심 기능
- 주거 선택에 필요한 정보 비교
- 조건 기반 탐색과 선택 지원
- 사용자 관점의 주거 정보 정리

## 기술 스택
- Java 17
- Spring Boot
- JPA
- PostgreSQL
- OpenLayers
- Docker

## 실행 환경
- Backend: Spring Boot
- Database: PostgreSQL

## 메모
이 저장소는 개인 프로젝트 코드 보관과 기능 실험 목적의 공개 저장소입니다.
