# HomeHarmony

아파트 선택 과정에서 입지와 생활 편의 요소를 함께 비교할 수 있도록 돕는 주거 선택 지원 프로젝트다.

## 프로젝트 목적
- 단순 매물 조회가 아니라 실제 거주 선택 기준에 가까운 정보를 한 번에 비교해보는 서비스를 만들고자 시작했다
- 교통, 편의시설, 생활 환경 같은 요소를 함께 보면서 의사결정을 돕는 방향을 목표로 했다

## 현재 저장소 상태
현재 공개 저장소에는 실행 가능한 애플리케이션 코드까지 정리되어 있지 않고,
프로젝트 아이디어와 방향을 보관하는 초기 단계 저장소에 가깝다.

즉 지금 기준으로는 완성된 서비스라기보다:
- 주거 선택 지원 서비스 아이디어
- 공간 정보 기반 비교 서비스 방향
- 이후 독립 서비스로 확장할 수 있는 후보 프로젝트

이렇게 보는 게 맞다.

## 구상한 핵심 기능
- 아파트 후보 비교
- 위치 기반 생활 편의 정보 조회
- 조건 기반 필터링
- 사용자 관점의 주거 선택 정보 정리

## 예상 기술 방향
- Backend: Spring Boot
- Data: PostgreSQL / PostGIS
- Frontend: 지도 기반 UI
- Infra: Docker

## 앞으로 정리할 것
- 실제 도메인 모델 정의
- 데이터 수집 범위 확정
- 지도 시각화 방식 결정
- MVP 기능 범위 축소

## 메모
이 저장소는 현재 “완성된 결과물”보다 “다음에 살릴 수 있는 프로젝트 아이디어”에 가깝다.
공개 저장소에서도 그 상태를 숨기지 않고 그대로 드러내는 편이 맞다.
